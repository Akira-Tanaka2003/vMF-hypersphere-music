# 成果物サマリー: vMF Hypersphere Music Representation

## 一言で説明

音楽のコード・root・5度圏上の近さを、vMF 分布の方向ベクトル \(\mu\) と角度で表現し、学習された超球面上に音楽理論的な構造が現れるかを可視化した研究成果物です。

## 何を作ったか

- MIDI 由来の音楽情報から、和声状態を vMF 超球面上の方向として表現
- コード種別ごとの中心方向を計算
- root ごとの中心方向を計算
- 方向ベクトル間の角度・cos 類似度を可視化
- 5度圏距離と vMF 角距離の対応を検証

## 主な結果

| 評価項目 | 結果 | 意味 |
|---|---:|---|
| Pearson 相関 | 0.954 | 5度圏距離と vMF 角距離が強く線形対応 |
| Spearman 相関 | 0.953 | 距離の順位関係も強く対応 |
| \(\mu\) 次元 | 8 | 学習された vMF 方向空間 |
| root クラス | 12 | C から B までの12音 |

## 面接・説明用の短い言い方

本研究では、音楽の和声を単なるラベルではなく、超球面上の方向として扱いました。  
その結果、学習された方向空間において、5度圏上で近い root ほど vMF 角距離も近くなる傾向が確認できました。

## 研究上の位置づけ

この成果物は、より大きな研究テーマである「vMF 超球面を利用した演奏生成」の可視化・検証部分です。  
最終的には、和声方向 \(\mu_t\)、集中度 \(\kappa_t\)、各音との方向一致度を用いて、velocity や伴奏表情を生成することを目指しています。

## 注意点

- root は pitch-class から推定したもので、完全な教師ラベルではありません。
- 3D 図は 8次元 \(\mu\) を PCA で可視化したものです。
- 公開リポジトリには、著作権上共有できない MIDI や学習済み重みは含めていません。
